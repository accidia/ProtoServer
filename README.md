## How to use?

Latest build: https://github.com/pteyer/jrz/archive/master.zip

Or use it as a maven dependency:

```xml
<dependencies>
    <dependency>
        <groupId>com.pteyer</groupId>
        <artifactId>jrz</artifactId>
        <version>0.0.1</version>
    </dependency>
</dependencies>
```
