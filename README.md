## How to use?

Latest build: https://github.com/accidia/jrz/archive/master.zip

Or use it as a maven dependency:

```xml
<dependencies>
    <dependency>
        <groupId>org.accidia</groupId>
        <artifactId>jrz</artifactId>
        <version>0.0.10</version>
    </dependency>
</dependencies>
```
